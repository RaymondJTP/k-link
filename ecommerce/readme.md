# Ecommerce simple API Documentation

## Execute first :

 - Run npm install on terminal
 - Atur config db di folder config/config.json
 - Run db:create
 - Run db:migrate
 - Run db:seed:all
 - 

## Endpoints :

List of available endpoints:

- `POST /register`
- `POST /login`
- `GET /products`
- `POST /products`
- `POST /products/:id`
- `POST /products/:id/payment`
- `GET /history`
